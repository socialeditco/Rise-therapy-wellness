# Rise Therapy & Wellness — Demo Redesign

A high-impact static demo/proposal site for Rise Therapy & Wellness.

## Build style
- Static HTML/CSS/JS for fast Vercel deployment.
- Warm, neuro-affirming visual system based on the provided Rise brand kit.
- Proposal-mode sections explain why each page module, widget, and form exists.
- Includes SEO/AEO-ready structure, JSON-LD, research proof points, and conversion-focused intake concepts.

## Included files
- `index.html` — full demo page and proposal content.
- `styles.css` — responsive design, animations, brand styling.
- `script.js` — mobile nav, tabs, intake wizard, FAQ accordion, modal page plans, before/after slider.
- `vercel.json` — static deployment headers and clean URLs.
- `assets/*.svg` — text-based logo/favicons created for the demo.

## Production recommendations
- Replace CSS illustrations with approved real Rise photography.
- Connect forms to approved HIPAA-conscious workflows or intake platforms.
- Add service detail routes/pages for OT, SLP, PT, Behavioral Health, Rise Sports, Rise-Ed, and Renovations.
- Add CMS for blog/events/resources if ongoing content updates are desired.
- Add GA4/GTM event tracking for call taps, portal clicks, intake starts, form completions, and downloads.
